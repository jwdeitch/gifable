POST /api/v1/gifs

Convert a GIF file into an HTML5 video. Capture a GIF via file upload as multipart/form-data then POST using the file parameter. The response to this POST request will contain basic file information, a link to the GIF, a link to a PNG poster generated from the GIF, and a link to the MP4 version.
