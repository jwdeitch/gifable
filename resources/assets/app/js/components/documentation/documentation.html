POST /api/v1/gifs

Convert a GIF file into an HTML5 video. Capture a GIF via file upload as multipart/form-data then POST using the file parameter. Transcoding the GIF is an asynchronous process, so the response to this POST request will only contain basic file information about the GIF and links to the CDN's HTTP and HTTPS versions of that file until transcoding has finished.
